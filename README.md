# HomeQuest -できた！を残す。-
## ～AI判定とAI演出を融合させた、自律型家事攻略RPG～

ユーザー一人一人がアカウントを所持

グループを作成(ホストになる) -> 家族をそこに招待

### ホスト側の操作
- 課題と報酬を設定
- 用意された課題を基にAIが毎回ランダムなクエストを生成
- - クエスト名
- - クエストの簡単なシナリオ
- - 演出
- 報酬 = 家庭内通貨
- - ホストが報酬ショップを設定し、報酬と交換できる

### プレイヤー側の操作
- ホストに招待されたグループに参加
- ホストが設定したクエストが発生すると通知が届く
- - クエスト(=家事)を完了したら写真をアップロード
- - ホストが判定(or AI判定)で合格が出ればクエストクリア
- - クオリティ不足で失敗 -> 即座にリベンジ可能
- - 制限時間内に家事を行わないとクエスト失敗

### その他の機能
- クエストクリアのAI判定
- - メジャーな家事についてはAIに完了した状態を覚えさせることでAIに判定させることができるようにする
- - ホストはクエスト設定時にAI判定をするか自身で判定するか選ぶ
- 報酬リクエストシステム
- - プレイヤーからほしい報酬と希望ポイントをホストに送る
- - ホストはそれを承認することで報酬ショップに追加できる

### 追加(2026/01/21)
- ユーザーは自身を識別するためのidと所属しているグループのid、ユーザー名、パスワードの4つの情報を持つ
- グループはそのグループを識別するためのid、そのグループに所属しているユーザーとその所持ポイント、登録されたクエストの情報、ホストが誰なのかの情報、報酬ショップの情報を持つ
- - 1つのグループにホストは複数人存在できる、最初にグループを作成した人はオーナーとなり、他のユーザーをホストに追加できるしくみ
